What is Lazy Loading

Lazy loading is the practice of delaying load or initialization of resources or objects until theyâ€™re actually needed to improve performance and save system resources. For example, if a web page has an image that the user has to scroll down to see, you can display a placeholder and lazy load the full image only when the user arrives to its location.
